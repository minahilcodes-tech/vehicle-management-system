#  Vehicle Management System

A console-based **C++** application developed using **Object-Oriented Programming (OOP)** concepts. The system allows users to manage vehicle records by adding, searching, updating, deleting, and storing data using file handling.

##  Features

- Add new vehicle records
- Display all vehicles
- Search vehicles by ID
- Update vehicle information
- Delete vehicle records
- Store data using text files

## Technologies

- C++
- Object-Oriented Programming (OOP)
- File Handling
- Visual Studio

##  How to Run

1. Open the project in Visual Studio.
2. Build and run the project.
3. Select an option from the menu.
4. Manage vehicle records.
